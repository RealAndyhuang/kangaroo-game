# 🦘 袋鼠沙漠大冒险 - 安装与运行说明

## 📋 系统要求

- **操作系统**: Windows 7/8/10/11 (64 位)
- **Python**: 3.8 或更高版本
- **内存**: 至少 512MB 可用内存
- **硬盘空间**: 约 100MB（打包后）

---

## 🚀 快速开始（推荐）

### 方法一：一键打包（Windows）

1. **下载并安装 Python**（如果还没安装）
   - 访问 https://www.python.org/downloads/
   - 下载 Python 3.10 或更高版本
   - ⚠️ **重要**：安装时勾选 "Add Python to PATH"

2. **双击运行 `build.bat`**
   - 脚本会自动安装依赖并打包
   - 等待 1-3 分钟

3. **运行游戏**
   - 打开生成的 `dist` 文件夹
   - 双击 `KangarooAdventure.exe`
   - 开始游戏！🎮

### 方法二：开发模式运行

1. 打开命令提示符（cmd）或 PowerShell
2. 进入游戏文件夹：
   ```cmd
   cd kangaroo-game
   ```
3. 安装依赖：
   ```cmd
   pip install -r requirements.txt
   ```
4. 运行游戏：
   ```cmd
   python kangaroo_game.py
   ```
   或双击 `run.bat`

## 🎮 游戏控制

| 按键 | 功能 |
|------|------|
| ← 或 A | 向左移动 |
| → 或 D | 向右移动 |
| 空格 或 W | 跳跃 |
| R | 重新开始当前关卡 |
| ESC | 退出游戏 |

## 📖 游戏玩法

### 目标
到达每关的**终点旗**（金色旗帜）即可进入下一关！

### 障碍物
- 🌵 **仙人掌**：碰到扣 20 血
- 🔴 **尖刺**：碰到扣 25 血
- 🕳️ **沙洞**：掉进去扣 30 血
- 🦂 **蝎子**：巡逻敌人，碰到扣 25 血
- 🐍 **蛇**：缓慢移动，碰到扣 20 血

### 道具
- 🍇 **仙人掌果**：吃到恢复 20 血

### 血量
- 初始血量：100
- 血量归零则游戏结束
- 按 R 可以重新开始当前关卡

## 📦 打包说明

### Windows 打包
```cmd
双击 build.bat
```

### macOS/Linux 打包
```bash
chmod +x build.sh
./build.sh
```

```cmd
pip install pyinstaller
pyinstaller kangaroo_game.spec
```

打包完成后，可执行文件在 `dist/` 文件夹中。

## 🔧 常见问题

### Q: 双击 build.bat 没反应？
A: 可能是 Python 没有添加到 PATH。请重新安装 Python，并确保勾选 "Add Python to PATH"。

### Q: 提示 "pygame not found"？
A: 运行以下命令安装依赖：
```cmd
pip install -r requirements.txt
```

### Q: 打包后文件太大？
A: 正常现象。打包包含 Python 运行时，所以文件较大（约 30-50MB）。可以使用 UPX 压缩（已在 spec 文件中启用）。

### Q: 游戏画面卡顿？
A: 确保你的电脑满足最低配置。关闭其他占用资源的程序。

### Q: 如何分享给朋友？
A: 将整个 `dist` 文件夹打包成 ZIP 发送，或者只发送 `KangarooAdventure.exe`（朋友需要确保系统有必要的运行库）。

## 🎯 关卡攻略

### 第一关：沙漠入门
- 教学关卡，熟悉跳跃和移动
- 注意避开仙人掌和第一个沙洞
- 吃到高台上的水果补充血量

### 第二关：危险沙丘
- 地面有很多断裂，注意不要掉下去
- 蝎子会来回巡逻，等它走开再通过
- 利用多层平台捷径

### 第三关：终极挑战
- 障碍密集，需要精准跳跃
- 蛇和蝎子混合巡逻
- 多吃水果保持血量健康
- 不要着急，稳扎稳打！
