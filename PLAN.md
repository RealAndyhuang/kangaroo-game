## 🎮 控制方案

### 单人模式
| 按键 | 功能 |
|------|------|
| ← → 或 A D | 移动 |
| 空格 或 W | 跳跃 |
| F 或 J | 出拳攻击 |
| P | 暂停 |
| R | 重新开始 |
| ESC | 返回菜单 |

### 双人模式（上屏 - 玩家 1）
| 按键 | 功能 |
|------|------|
| A D | 移动 |
| W | 跳跃 |
| F | 攻击 |

### 双人模式（下屏 - 玩家 2）
| 按键 | 功能 |
|------|------|
| ← → | 移动 |
| ↑ | 跳跃 |
| Enter | 攻击 |

## 🔧 技术要点

1. **中文显示**: 使用 `pygame.font.SysFont('simhei', size)` 或加载中文字体文件
2. **分屏渲染**: 使用 `pygame.Surface` 创建两个视口，分别渲染
3. **精灵动画**: 使用序列帧 PNG 图片
4. **音频**: `pygame.mixer` 模块
5. **攻击判定**: 矩形碰撞 + 攻击帧检测
