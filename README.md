# 🦘 袋鼠沙漠大冒险 3.0 - 街霸精美版

## 游戏简介

一款**街霸级别精美**的 2D 平台动作游戏！扮演澳洲袋鼠在沙漠中冒险，用拳击打败敌人，收集水果，挑战 15 个精心设计的关卡！

---

## ✨ 游戏特色

### 🎨 精美美术
- **程序化精美绘制** - 无需外部图片，所有精灵图均为程序化绘制
- **渐变色彩** - 角色和场景使用丰富的渐变效果
- **动画流畅** - 每个角色都有多帧动画
- **视差背景** - 三层视差滚动沙漠场景

### 🥊 拳击战斗系统
- **出拳攻击** - 按 F/J (玩家 1) 或 Enter (玩家 2) 出拳
- **敌人血量** - 多次攻击可消灭敌人
- **攻击特效** - 击中时有星星特效
- **跳跃攻击** - 跳起来打鸟、打蘑菇

### 🎮 双人分屏模式
- **上下分屏** - 真正的独立分屏，不是同屏
- **独立操作** - 修复了操作同步 bug
- **每玩家 3 条命** - 独立生命系统

### 🎵 音效系统
- **背景音乐** - 可选开关
- **游戏音效** - 跳跃、攻击、击中、收集、敌人死亡
- **菜单音效** - 选择、确认音效

### 📋 完整菜单系统
- **单打模式** - 单人冒险
- **双打模式** - 本地双人合作
- **设置选项** - 音效/音乐开关
- **退出游戏**

### 🌵 丰富障碍物
- **仙人掌** - 经典沙漠障碍
- **飞鸟** - 水平飞行敌人
- **老鹰** - 会俯冲攻击，还会...拉屎！💩
- **毒蘑菇** - 左右移动的邪恶蘑菇
- **火车** - 移动平台，会扔垃圾/瓶子/香蕉
- **果树** - 可以跳起来吃果子

## 🎮 控制方式

### 单人模式
| 按键 | 功能 |
|------|------|
| ← → 或 A D | 移动 |
| 空格 或 W | 跳跃 |
| F 或 J | 出拳攻击 |
| P | 暂停 |
| R | 重新开始 |
| ESC | 返回菜单 |

### 双人模式 - 玩家 1 (上屏)
| 按键 | 功能 |
|------|------|
| A D | 移动 |
| W | 跳跃 |
| F | 攻击 |

### 双人模式 - 玩家 2 (下屏)
| 按键 | 功能 |
|------|------|
| ← → | 移动 |
| ↑ | 跳跃 |
| Enter | 攻击 |

## 🚀 快速开始

### 方法 1: 直接运行 EXE (推荐)
1. 下载 `KangarooAdventure3.zip`
2. 解压到任意文件夹
3. 双击 `KangarooAdventure3.exe`

### 方法 2: 开发模式运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python kangaroo_game_3.py
```

### 方法 3: 自己打包
**Windows:**
```bash
build.bat
```

**Linux/Mac:**
```bash
chmod +x build.sh
./build.sh
```

## 🎯 游戏目标

1. **到达终点** - 每关都有终点旗帜
2. **消灭敌人** - 用拳击打败鸟、蘑菇、老鹰

## ⚠️ 注意事项

### 中文显示
游戏已修复中文乱码问题，会自动选择系统中文字体。如果仍显示方块，请确保系统已安装以下字体之一：
- 黑体 (simhei)
- 宋体 (simsun)
- 微软雅黑 (microsoft yahei)
- 文泉驿正黑 (wqy-zenhei)

### 系统要求
- **操作系统**: Windows 7/8/10/11 或 Linux/Mac
- **Python**: 3.8+ (开发模式需要)
- **内存**: 512MB
- **硬盘**: 100MB (打包后)
- **显卡**: 支持 OpenGL 2.0+

### 音效
游戏使用程序化生成音效，无需外部音频文件。如果听不到声音，请检查：
1. 系统音量
2. 游戏设置中的音效开关
3. 扬声器/耳机连接
